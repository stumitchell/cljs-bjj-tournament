About & Licence

About
Score Master is a simple, open source scoring application for use with IBJJF competition rules.

It has been ported from the original Python3 / Gtk3 version as a programming challenge and to make it truly cross-platform.

This software was written for use on pcs with a screen resolution of 1366x768.

You can check my website for additional Jiu-Jitsu applications. http://www.slumbergod.com

Licence
This version of Score Master is provided on a free-to-use and free-to-modify licence. It is provided as-is and the author assumes no responsibility for any problems you encounter in its usage.

If you make modifications you must leave the information on this About & Licence page intact but you can add yourself to the credits column of this page.

You must release your modified version under the same licence.

Customisation
This software was written for use within the organisation I train with. For this reason it is branded myBJJ.

You are welcome to rebrand it however you like, provided you satisfy the licence conditions.

Like most open source software projects, it just isn't feasible to provide free support. If you are unable to rebrand, customise, or extend the application yourself I can do it for you for a small fee.

If you find the software useful and want to ensure its future availability and development please consider making a donation.

Those who have contributed financially will have their version upgraded from the basic version to the development version which includes additional features and improvements as they are added.

You can contact me from my website: http://www.slumbergod.com
